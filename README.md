# ``BbAppHttpClient``

HTTP client for network communication.

## Overview

The BbAppHttpClient library provides a robust HTTP client for all network communication needs. It includes request building utilities, response handling, error management, authentication header injection, and network connectivity monitoring. This client serves as the foundation for all API communication throughout the application, providing a consistent interface for HTTP operations.
